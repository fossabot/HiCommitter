HiCommitter
===========

HiCommitter

在化肥作为committer, 每次帮人提交代码都要清理一遍SVN账号.
在设置中点击清除比较麻烦, 提交完成后dev需要再次输入自己的账号也很麻烦.
使用这个程序, 一键清除保存的账号(备份起来), 提交完成后再一键还原账号.
